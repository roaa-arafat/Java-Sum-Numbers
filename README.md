# Java-Sum-Numbers
A Java program that calculates the sum of odd, even, prime numbers, or a combination of odd and prime numbers based on user input.

This Java program asks the user to enter a series of numbers and then calculates:
- Sum of odd numbers
- Sum of even numbers
- Sum of prime numbers
- Sum of numbers that are both odd and prime

## How to Run
1. Compile the program: `javac w3_task1.java`
2. Run the program: `java w3_task1`
3. Follow the prompts
